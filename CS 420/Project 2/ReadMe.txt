To run:

Inside the terminal, navigate to the "Leach_Genevieve_420p2/src" folder.
In the terminal, compile the program using the command "javac NQueensDriver.java",
and then run using the command "java NQueensDriver".

If you want to change the amount of problems, locate the NQueensDriver.java file and edit the constant NUMPROBLEMS.

The solution for each problem is shown as an array, with the number of each indicating a row within that column that the Queen resides.
